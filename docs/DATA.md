# Data

## CIFAR-10 (reproducible experiments)
CIFAR-10 is obtained **automatically** via `torchvision.datasets.CIFAR10`
(`download=True`) the first time you run any CIFAR-10 script. The expected data
directory is `./data` relative to the artifact root (pass `--data-dir data`,
which the provided scripts already do). The dataset is ~170 MB and is not
shipped with this artifact.

No manual download is required; an internet connection is needed on first run.

## ImageNet-100 (implementation/config only)
The ImageNet-100 experiments expect a dataset directory (the code default is
`data/imagenet100`). ImageNet-100 is **not** included in this artifact for size
and licensing reasons. ImageNet-100 is a standard 100-class subset of ImageNet;
obtain ImageNet from its official source and arrange the 100-class subset in the
standard `train/` and `val/` ImageFolder layout under `data/imagenet100`.

## Multimodal (BLIP-2 / LLaVA)
The multimodal susceptibility study used a public image-caption subset
(COCO-Caption2017). Only the resulting small metric tables are included
(`results/other_experiments/multimodal/`); no images are shipped.

## What is intentionally NOT included
- Large raw datasets (CIFAR-10 binaries, ImageNet-100 images).
- Trained model checkpoints / activation tensors / correlation matrices
  (regenerated by running the pipeline).
